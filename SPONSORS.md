# Sponsors

Mycelium is MIT-licensed and developed openly. We are sustained by sponsors
who believe code intelligence should be local, fast, and accessible.

## How to Sponsor

- **[GitHub Sponsors](https://github.com/sponsors/aimasteracc)** — recurring or one-time
- **OpenCollective** — TBD
- **Buy Me a Coffee** — TBD

## What Sponsorship Funds

In order of priority:

1. **AI API and compute** for the Hive (currently covered by Claude Code Max 20×, but at scale we will need more).
2. **CI minutes and runners** for cross-platform builds and nightly benchmarks.
3. **Domain, docs hosting, code signing certificates**.
4. **Direct compensation** for significant non-BDFL human contributors (transparent disbursement quarterly).
5. **Design and branding work** (logo refresh, docs theme, swag for top contributors).

Quarterly disbursement reports are posted at the bottom of this file.

## Ethics

Per [GOVERNANCE.md](GOVERNANCE.md), the project does not accept funding
from entities the BDFL judges to pose a conflict of interest with the
project's open-source mission. The current policy is conservative — when
in doubt, decline. If a declined offer becomes notable, it will be
disclosed here with the reasoning.
